## Getting started

* git clone --recurse-submodules https://github.com/ht-diva/pqtl_pipeline.git
* cd pqtl_pipeline
* adapt the [submit.sbatch](submit.sbatch) and [config/config.yaml](config/config.yaml) files to your environment
* sbatch submit.sbatch
